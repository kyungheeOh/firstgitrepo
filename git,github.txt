git bash - bash sell(linux)환경 구성, linux명령어 사용
ex)
pwd : 현재 작업중인 경로 표시
cd : 경로변경 
ls : 모든 파일 리스트
mkdir : 새로운 디렉토리 생성

vi : 문서 파일 편집기
i insert mode(현재 커서 위치) <> esc normal mode
a append mode(현재 커서 위치 + 한칸) 
o open line mode(현재 커서 아랫줄)
:w 저장 :q 종료 :q! 강제종료 :wq 저장종료 :wq! 강제저장종료
h,j,k,l 커서상하좌우(=방향키)
